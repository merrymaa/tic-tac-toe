запрос на регистрацию нового пользователя:
curl -X POST http://localhost:5000/register \
-H "Content-Type: application/json" \
  -d '{
  "login": "evg",
  "password": "pa3453454"
}'

преобразование пары логин:пароль в формат base64:
echo -n "evg:pa3453454" | base64
ZXZnOnBhMzQ1MzQ1NA==

$ echo -n "gamer:pa3453454" | base64
Z2FtZXI6cGEzNDUzNDU0



запрос на авторизацию пользователя (пользователь существует):
curl -X POST http://localhost:5000/authorize \
   -H "Authorization: Basic Z2FtZXI6cGEzNDUzNDU0"

запрос на авторизацию пользователя (пользователь не существует):
curl -X POST http://localhost:5000/authorize \
   -H "Authorization: Basic ZXZnXzI6cGEzNDUzNDU0"

запрос на создание новой игры (с компьютером):
curl -X POST http://localhost:5000/new_game \
  -H "Authorization: Basic ZXZnOnBhMzQ1MzQ1NA==" \
  -H "Content-Type: application/json" \
  -d '{
  "game_type" : "AI"
}'

запрос на создание новой игры (с человеком):

  -H "Content-Type: application/json" \
